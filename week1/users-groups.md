# Users & Groups (Week 1)

## Key commands
- `whoami` / `id`
- `adduser` / `useradd`
- `passwd`
- `groups` / `getent group`
- `usermod -aG <group> <user>`
- `sudo` basics

## Notes
- Users have UID; groups have GID
- Add user to group: `usermod -aG`
